# Instructions pour ajouter les images d'équipements Aywa

## Système d'images configuré
La base de données PostgreSQL est configurée avec des chemins d'images locaux pour chaque équipement.

## Noms de fichiers requis
Placez vos vraies photos d'équipements dans ce dossier avec ces noms exacts :

1. **generateur-100kva.jpg** - Générateur électrique 100 KVA
2. **pelleteuse-20t.jpg** - Pelleteuse hydraulique 20 tonnes  
3. **camion-citerne-10000l.jpg** - Camion-citerne 10 000L
4. **motopompe-haute-pression.jpg** - Motopompe irrigation
5. **compresseur-500l.jpg** - Compresseur 500L
6. **echafaudage-mobile.jpg** - Échafaudage aluminium mobile
7. **compresseur-air-500l.jpg** - Compresseur d'air 500L
8. **chargeuse-pneus.jpg** - Chargeuse sur pneus
9. **betonniere-350l.jpg** - Bétonnière 350L
10. **foreuse-eau-mobile.jpg** - Foreuse à eau mobile
11. **camion-grue-25t.jpg** - Camion grue 25 tonnes
12. **generateur-solaire-50kva.jpg** - Générateur solaire 50 KVA
13. **tracteur-agricole-85cv.jpg** - Tracteur agricole 85 CV
14. **sonorisation-5000w.jpg** - Système sonorisation 5000W
15. **station-lavage-mobile.jpg** - Station lavage mobile

## Format recommandé
- Format : JPG ou PNG
- Taille : 800x400 pixels minimum
- Qualité : Haute résolution pour un affichage professionnel

## Accès aux images
Une fois ajoutées, les images seront accessibles via :
`https://votre-domaine.com/images/nom-fichier.jpg`

## Base de données
La base de données PostgreSQL stocke les chemins d'images et toutes les informations des équipements.
