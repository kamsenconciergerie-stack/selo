# Variations du Logo Kamsen pour Réseaux Sociaux

## Logo Principal Analysé

**Fichier source :** `84129083_1753626050381.png`

**Caractéristiques :**
- Format horizontal rectangulaire
- Couleurs : Bleu (#4A90E2), Orange (#FF6B35), Blanc
- Composition : Icône chargeuse/grue + texte "KAMSEN" + slogan "DES CHARGES EN MOINS"
- Style : Moderne, professionnel, industriel

## Adaptations Nécessaires pour Réseaux Sociaux

### 1. Logo Carré (Instagram/Facebook profil)
**Dimensions :** 320x320px
**Description :**
- Version simplifiée avec uniquement l'icône chargeuse/grue
- Fond bleu Kamsen avec icône orange
- Texte "KAMSEN" en bas en blanc
- Sans le slogan pour lisibilité

### 2. Logo Horizontal (Couvertures Facebook)
**Dimensions :** 820x312px  
**Description :**
- Logo complet actuel adapté
- Fond transparent ou blanc
- Taille optimisée pour couverture Facebook

### 3. Logo Vertical (Stories Instagram)
**Dimensions :** 1080x1920px
**Description :**
- Icône en haut
- "KAMSEN" au centre
- "DES CHARGES EN MOINS" en bas
- Format portrait pour stories

### 4. Logo Watermark (Transparent)
**Dimensions :** 200x50px
**Description :**
- Version mini horizontale
- Opacité 30% blanc avec ombre
- Pour marquer les photos d'équipements

## Couleurs Précises Extraites

**Bleu Principal :** #398BDE (dérivé du bleu visible)
**Orange Accent :** #FF6B35 (orange du texte KAMSEN)
**Bleu Slogan :** #4A7DC4 (bleu de la barre slogan)
**Blanc :** #FFFFFF

## Instructions de Création

### Logo Carré (Profil)
```
1. Prendre l'icône chargeuse/grue uniquement
2. Redimensionner pour s'adapter au carré
3. Fond : Bleu #398BDE
4. Icône : Orange #FF6B35
5. Texte "KAMSEN" en bas : Blanc #FFFFFF
6. Police : Inter Bold ou similaire
```

### Logo Stories
```
1. Format vertical 9:16
2. Icône chargeuse centrée haut (30% hauteur)
3. "KAMSEN" centré milieu (25% hauteur)
4. "DES CHARGES EN MOINS" centré bas (15% hauteur)
5. Fond : Gradient bleu ou blanc selon usage
```

### Logo Watermark
```
1. Version horizontale miniature
2. Tous éléments en blanc avec ombre portée
3. Opacité 30% pour discrétion
4. Position coin inférieur droit des photos
```

**Site Web :**

## Cohérence Marque

**À respecter :**
- Toujours utiliser les couleurs exactes Kamsen
- Maintenir les proportions icône/texte
- Fond blanc ou bleu uniquement (pas d'autres couleurs)
- Police cohérente (Inter ou équivalent)

**À éviter :**
- Déformation du logo
- Couleurs différentes
- Ajout d'éléments graphiques
- Utilisation sur fonds colorés non-Kamsen

Le nouveau logo haute qualité est maintenant intégré dans le header du site et prêt pour déclinaison réseaux sociaux !
